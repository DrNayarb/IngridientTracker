Ingredients Tracker

Instructions:

If this is your first time using this webiste, you can add a food by entering a name, using the calender to enter a date, and place a number for quantity. Afterward, click the save button to save your information as a csv file. If you have already created a csv file, click on the choose a csv file button and click on the previously stored csv. The bottom textarea shows expiration dates of the selected ingridient. You may delete an ingrideint with the delete buttom. Lastly, you may use an ingridient by using the use button.
